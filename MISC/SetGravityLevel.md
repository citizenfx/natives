## SET_GRAVITY_LEVEL

```c
// 0x740E14FAD5842351 0x2D833F4A
void SET_GRAVITY_LEVEL(int level);
```

```
This native adjusts the in-game gravity level by the given amount.
```

```
Level can be set inbetween 0-3
0: 9.8   
1: 2.4   
2: 0.1 - very low  
3: 0.0
```

* **level**:
