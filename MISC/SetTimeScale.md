## SET_TIME_SCALE

```c
// 0x1D408577D440E81E 0xA7F84694
void SET_TIME_SCALE(float timeScale);
```

```
Maximum value is 1.  
At a value of 0 the game will still run at a minimum time scale.  
Slow Motion 1: 0.6  
Slow Motion 2: 0.4  
Slow Motion 3: 0.2  
```
