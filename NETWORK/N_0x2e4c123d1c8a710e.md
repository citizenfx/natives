## NETWORK_IS_ANY_PLAYER_NEAR
```c
// 0x2E4C123D1C8A710E 0x0955C6F927712241
BOOL NETWORK_IS_ANY_PLAYER_NEAR(int* retPlayerIds, int* retNumber, float x, float y, float z, float radius, BOOL p6);
```

Returns whether any networked players are near the specified coordinates, and outputs their IDs.

## Parameters
* **retPlayerIds**: Pointer to an array that receives the Player IDs of nearby players.
* **retNumber**: Pointer to receive the number of nearby players found.
* **x**: X coordinate of the search position.
* **y**: Y coordinate of the search position.
* **z**: Z coordinate of the search position.
* **radius**: The search radius around the position.
* **p6**: Unknown. Possibly a filter flag (e.g., exclude self).

## Return value
`TRUE` if any player is found within the radius, `FALSE` otherwise.
