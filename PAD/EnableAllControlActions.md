## ENABLE_ALL_CONTROL_ACTIONS

```c
// 0xFC2F119F
void ENABLE_ALL_CONTROL_ACTIONS(int controlType);
```

```
controlType:
PLAYER_CONTROL = 0
CAMERA_CONTROL = 1
FRONTEND_CONTROL = 2
```

## Parameters
* **controlType**: The type of control inputs.

Enables all the inputs for a player control.
