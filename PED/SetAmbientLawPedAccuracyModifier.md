## SET_AMBIENT_LAW_PED_ACCURACY_MODIFIER

```c
// 0x87DDEB611B329A9C
void SET_AMBIENT_LAW_PED_ACCURACY_MODIFIER(float multiplier);
```

Function must be called every frame.

## Parameters
* **multiplier**: Multiplier to apply to the accuracy of all Law Peds.
