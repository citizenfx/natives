## IS_PED_GESTURING

```c
// 0xC30BDAEE47256C13
BOOL IS_PED_GESTURING(Ped ped);
```

Checks if a ped is gesturing.

```
NativeDB Introduced: v1868
```

## Parameters
* **ped**: Ped that is being checked for gesturing

## Return value
Returns true if ped is gesturing. Otherwise false.
