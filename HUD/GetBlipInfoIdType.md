## GET_BLIP_INFO_ID_TYPE

```c
// 0xBE9B0959FFD0779B 0x501D7B4E
int GET_BLIP_INFO_ID_TYPE(Blip blip);
```

```
Returns a value based on what the blip is attached to
1 - Vehicle
2 - Ped
3 - Object
4 - Coord
5 - unk
6 - Pickup
7 - Radius
```
