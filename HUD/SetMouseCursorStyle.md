## SET_MOUSE_CURSOR_STYLE

```c
// 0x8DB8CFFD58B62552
void SET_MOUSE_CURSOR_STYLE(int style);
```

Changes the mouse style.

Available styles are:
| ID  | Description         |
| :-: | :-----------------: |
| 0   | Normal              |
| 1   | Normal              |
| 2   | Transparent         |
| 3   | Pre-Grab            |
| 4   | Grab                |
| 5   | Click/Middle Finger |
| 6   | Left Arrow          |
| 7   | Right Arrow         |
| 8   | Up Arrow            |
| 9   | Down Arrow          |
| 10  | Horizontal Expand   |
| 11  | Add                 |
| 12  | Remove              |

## Parameters
* **style**: Which sprite to change to
