## PAUSE_MENU_GET_MOUSE_HOVER_UNIQUE_ID

```c
// 0x13C4B962653A5280
int PAUSE_MENU_GET_MOUSE_HOVER_UNIQUE_ID();
```

## Return value
Returns the uniqueID of the pause menu item the mouse is hovering on. -1 if the mouse isn't rolled over any item.
