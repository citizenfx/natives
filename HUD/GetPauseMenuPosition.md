## GET_PAUSE_MENU_POSITION

```c
// 0x5BFF36D6ED83E0AE
Vector3 GET_PAUSE_MENU_POSITION();
```


## Return value
This is used as a vector2, so the z parameter is always 0.
Returns starting position coordinates of the Pause Menu's body component.
