## SET_BLIP_ALPHA

```c
// 0x45FF974EEE1C8734 0xA791FCCD
void SET_BLIP_ALPHA(Blip blip, int alpha);
```

```
Sets alpha-channel for blip color.
Example:
Blip blip = HUD::ADD_BLIP_FOR_ENTITY(entity);
HUD::SET_BLIP_COLOUR(blip , 3);
HUD::SET_BLIP_ALPHA(blip , 64);
```
